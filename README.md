# WebAssembly C++ Calculator

A simple, lightweight calculator application that runs C++ logic in the browser. This project uses Emscripten to compile C++ arithmetic functions into WebAssembly (WASM), allowing the JavaScript frontend to call backend logic directly without using a server.

## ğŸš€ How It Works

1.  **Logic**: Core arithmetic is written in C++ header files (`add.h`, `subtract.h`, etc.).
2.  **Bridge**: `wasm_bridge.cpp` imports these headers and exposes specific API functions to the web environment.
3.  **Compilation**: The `build.sh` script uses Emscripten to compile the C++ code into a `.wasm` binary and a `.js` glue file.
4.  **Frontend**: Standard HTML/JS interacts with the compiled code using `ccall` or `cwrap`. **No HTML5 Canvas is used; this operates purely on DOM elements.**

This will generate `public/calculator.js` and `public/calculator.wasm`.

### The Build Command
The project uses the following `emcc` configuration (found in `build.sh`):

```bash
emcc cpp/wasm_bridge.cpp -o public/calculator.js \
  -I./cpp \
  -s EXPORTED_FUNCTIONS='["_api_add", "_api_subtract", "_api_multiply", "_api_divide"]' \
  -s EXPORTED_RUNTIME_METHODS='["ccall", "cwrap"]' \
  -s WASM=1
```
  ## ğŸƒâ€â™‚ï¸ Running the App

Because WebAssembly modules are loaded externally, browsers will block them if you try to open `index.html` directly via the file system (due to CORS policies).

**You must use a local web server.**

### Using Python 3
```bash
# Run this from the project root
python3 -m http.server 8080 --directory public
```
###Using Node (http-server)
```bash
npx http-server public
```

Open http://localhost:8080 in your browser to use the calculator.
